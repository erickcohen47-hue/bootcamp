# bootcamp
Simulador Investimentos Fundos Imobiliarios


O que o projeto faz
Simula resultados de como você pode investir trazendo valores reais a partir de informações básicas

Por que o projeto é útil
Ajuda a pessoas que não tem noção de invetimentos a darem seus primeiros passos 

Como os usuários podem começar com o projeto
Como um mínimo conhecimento de EXCEL, qualquer pessoa poderá explorar o APP e obter resultados a partir dos dados informados

Onde os usuários podem obter ajuda com seu projeto
Em caso de dúvidas podem usar minhas redes sociais e encaminhar suas dúvidas. As perguntas podem levar até 1 dia para serem respondidas

Quem mantém e contribui para o projeto
O código está aberto e toda colaboração é sempre bem vinda. No entanto, descreva quais implementações foram adicionadas para que futuros usúarios possam entende-lás.
